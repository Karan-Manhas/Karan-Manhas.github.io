About Me

Hi there, I’m Karan, a Computer Scientist passionate about building intelligent systems and scalable solutions. My work combines Machine Learning, Artificial Intelligence, and Data Engineering to solve real-world problems with pragmatic code.

I’ve developed deep learning models for tasks like age and gender classification using ResNet50, and built spam classifiers with Naive Bayes. The data engineering experience includes orchestrating serverless ETL pipelines with AWS Lambda, modeling dimensional data warehouses using Kimball methodology, and creating real-time dashboards with Redshift, Flask, and Docker.

I specialize in backend and pipeline development, having built robust data workflows with Apache Airflow, integrated APIs for large-scale ingestion, and automated pipelines to improve operational efficiency. My most recent is on **Post Quantum Cryptography** where I benchmarked algorithms like KYBER-CRYSTALS and ECDH in distributed systems using C++, OpenSSL, and liboqs on AWS.
